Generar Reporte de Inventario

Sube un archivo CSV que contenga códigos de barras para generar un reporte detallado con las cantidades y otra información de cada producto.

El archivo debe contener una columna con códigos de barras. Cada línea representa una unidad del producto.
Instrucciones
Formato del archivo:
  • Archivo CSV con una columna de códigos de barras
  • Cada línea representa una unidad del producto
  • Si hay códigos repetidos, se contarán como múltiples unidades
  • No necesita tener encabezados
Contenido del reporte:
  • Total de productos diferentes
  • Cantidades de cada producto
  • Nombres de productos (obtenidos desde Odoo)
  • Precios unitarios y valores totales
  • Listado de productos no encontrados en Odoo
Nota: Este reporte utiliza los datos de tu instancia de Odoo para obtener los nombres y precios de los productos. Asegúrate de que la configuración de conexión sea correcta.
